一、项目名称：
        国美电商平台

二、使用到的技术栈：
        1.使用了node搭建后台，配合本地存储技术实现登陆注册功能和购物车功能。
        2.整个项目的商品数据全部采用json数据托管，使用ajax数据请求技术获取json文件，进行数据的解析和动态渲染商品信息。
        3.使用了jQuery 实现交互功能。
        4.使用git托管整个项目文件。
        5.项目采用了模块化开发

三、使用方式：
        1.使用命令窗口进入project
        2.node server
        3.到浏览器输入：localhost:3000/homePage.html,可以打开首页
        4.页面实现交互部分：
            - 首页：页面的的头部、搜索框、三级菜单、轮播图、楼层、商品和选项卡；
            - 商品详情页面：页面头部、搜索框部分、导航、小图切换和右侧详情；
            - 商品列表页面：页面头部、搜索框部分、单个商品部分
            - 购物车页面：页面头部、商品部分
            - 登陆注册页面：登陆注册部分
        5.项目完成功能：
            - 登陆守卫
            - 登陆注册、表单正则验证
            - 轮播图
            - 三级菜单
            - 倒计时
            - 商品选项卡切换
            - 商品详情渲染
            - 添加购物车
            - 购物车的全选单选价格的计算
            - 放大镜功能
            - 小图切换大图功能
            - ajax商品分页渲染
        6.项目说明：
            - 后续使用gulp工具进行项目的压缩和转换
            - 全项目更改模块化
            - git版本管理实时上传
